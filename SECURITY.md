# Security Policy

Nie publikuj zgłoszeń zawierających dane logowania, VIN, współrzędne pojazdu ani pełne raporty produkcyjne.

W przypadku przypadkowego ujawnienia sekretu:

1. natychmiast zmień hasło lub unieważnij token,
2. usuń sekret z repozytorium i historii Git,
3. sprawdź logi dostępu do powiązanych usług.
